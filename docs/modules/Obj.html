getIn

  • getIn(obj: any, ...keys: (string | number)[]): any | null
    • Defined in object.ts:29

    Retrieves data deeply nested within an object.

    This prevents having to check for null or undefined values at each nested level of the object.

    Parameters

    • obj: any

      the object containing the data to retrieve

    • Rest ...keys: (string | number)[]

      an argument list of keys

    Returns any | null

    Usage:

    const object = {
  a: {
    b: {
      c: "Hello!"
    }
  }
}

Obj.getIn(object, "a", "b", "c")     // "Hello!"
Obj.getIn(object, "x", "y")          // null
